# ConveniencePayService
## 프로젝트 설명
<p>편의점 결제 서비스(일명 ’편결이')</p>
<p>사용자가 편의점에서 간편결제로 결제할 수 있는 시스템 개발</p>
<p>사용자 → 편결이 → 결제수단 시스템(이미 만들어져 있음) 호출</p>
